# Getting Started

This guide will help you get up and running with Prompt My Client React in your project.

## üìã Prerequisites

Before you begin, make sure you have:

- Node.js 16.8 or later
- A React project (version 18 or higher)
- npm, yarn, or pnpm package manager

## üöÄ Installation

### Using npm

```bash
npm install prompt-my-client-react
```

### Using yarn

```bash
yarn add prompt-my-client-react
```

### Using pnpm

```bash
pnpm add prompt-my-client-react
```

## üîß Basic Setup

### 1. Import the Component

```tsx
import { AIDropDown } from 'prompt-my-client-react';
```

### 2. Add to Your Component

```tsx
import { usePrompt } from 'prompt-my-client-react';

function App() {
  const [input, setInput] = useState('');
  
  const { generatePrompt, isLoading, error, result } = usePrompt({
    onSuccess: (response) => {
      setInput(response);
      console.log('AI Response:', response);
    },
    onError: (error) => {
      console.error('Error:', error);
    }
  });

  const handleGenerate = async () => {
    await generatePrompt(
      input,
      "Make this text more professional and concise",
      "professional",
      "concise"
    );
  };

  return (
    <div>
      <textarea 
        value={input} 
        onChange={(e) => setInput(e.target.value)}
        placeholder="Type your text here..." 
      />
      <button onClick={handleGenerate} disabled={isLoading}>
        {isLoading ? 'Processing...' : 'Make Professional'}
      </button>
      {error && <p>Error: {error}</p>}
      {result && <p>Result: {result}</p>}
    </div>
  );
}
```

## üîç What Happens Next?

1. **User Types Text**: The user enters text in the textarea
2. **AI Button Appears**: The AI dropdown button is positioned next to the text
3. **User Clicks AI Button**: A dropdown menu appears with various text transformation options
4. **AI Processes Request**: The selected prompt is sent to the AI service
5. **Response Received**: The transformed text is returned via the callback function
6. **UI Updates**: The textarea is automatically updated with the new content

## üö® Common Issues

### Component Not Rendering

- Make sure you've imported the component correctly
- Check that your React version is 18 or higher
- Verify the package was installed successfully

### AI Responses Not Working

- Ensure the `text` prop contains the text you want to transform
- Check that the `aiResponseCallback` function is properly defined
- Verify your internet connection for AI service access
