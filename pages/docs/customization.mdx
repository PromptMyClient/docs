### Button Size

The component supports three predefined button sizes:

```tsx
import { AIDropDown } from 'prompt-my-client-react';

// Small button (38px) - Good for compact interfaces
<AIDropDown 
  text={text} 
  aiResponseCallback={handleAIResponse}
  size="38px"
/>

// Medium button (48px) - Default size
<AIDropDown 
  text={text} 
  aiResponseCallback={handleAIResponse}
  size="48px"
/>

// Large button (52px) - Good for prominent placement
<AIDropDown 
  text={text} 
  aiResponseCallback={handleAIResponse}
  size="52px"
/>
```

## ğŸ¯ Best Practices

### Consistency

- Use consistent color schemes across your application
- Maintain consistent spacing and sizing
- Follow your design system's guidelines

### Accessibility

- Ensure sufficient color contrast
- Provide clear visual feedback on interactions
- Support keyboard navigation

### Performance

- Avoid complex animations on mobile devices
- Use CSS transforms instead of changing layout properties
- Minimize repaints and reflows
